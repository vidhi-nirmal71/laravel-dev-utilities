This document outlines the detailed steps required to upgrade a Laravel project from version 10.47 to 11.

1. Backup the Project and Database

cp -r laravel-adminpanel laravel-adminpanel-backup

# OR use Git
git add .
git commit -m "Backup before Laravel 11 upgrade"
git tag pre-laravel-11
git push origin dev-upgrade-10-to-11 --tags

# Back up the database
mysqldump -u youruser -p yourdatabase > db_backup.sql

2. Update Laravel Version in composer.json

Edit composer.json: "laravel/framework": "^11.0"


composer update
COMPOSER_MEMORY_LIMIT=-1 composer update

3. Fix Package Compatibility Issues

  • Problem: Package requires laravel/framework ^10.47
  • Solution: composer require vendor/package:^X.Y

4. Clear and Optimize Laravel Setup


php artisan config:clear
php artisan route:clear
php artisan view:clear
php artisan cache:clear
composer dump-autoload
php artisan optimize

5. Handle Code Changes (Handler.php)

  • Adjust method signatures
  • Remove deprecated methods
  • Add missing imports

6. Check Project Functionality

Ensure:

  • Pages load correctly
  • JS and CSS are working
  • No console or network errors

7. Run Migrations and Test Data


php artisan migrate
php artisan db:seed

8. Run Tests

php artisan test

9. Final Cleanup


php artisan config:cache
php artisan route:cache
php artisan view:cache

10. Summary of Commands


# Backup
cp -r laravel-adminpanel laravel-adminpanel-backup
mysqldump -u root -p dbname > backup.sql

# Update
composer update
COMPOSER_MEMORY_LIMIT=-1 composer update

# Fix packages
composer require vendor/package:^version

# Clear + optimize
php artisan config:clear
php artisan optimize
php artisan migrate
php artisan serve
php artisan test

# Commit
git commit -am "Upgraded Laravel to 11.00"
git push
